## Detailed Summary

Bu source `Rust for Backend Developers` kitobining maqsadini belgilaydi: boshqa tillarda tajribasi bor developerlar uchun Rustda back-end application yozishga tez kirish. Kitob avval Rust tilining o'zini, keyin [[cargo|Cargo]], server-side development uchun kerakli components va patterns, undan keyin [[async-await|async/await]] va to'liq back-end application yaratishni ko'rib chiqadi.

Target audience - Java, C#, Python, Ruby yoki Go bilan ishlagan back-end engineerlar. Muallif ayniqsa performance yoki resource usage bo'yicha mavjud tildan norozi bo'lgan developerlarni nazarda tutadi. Bu Rustni "birinchi programming language" sifatida emas, balki mavjud engineering tajribasi ustiga o'rganiladigan system/application language sifatida frame qiladi.

Kitob umumiy programming tushunchalarini juda qisqa ko'rib chiqadi: primitive types, variables, `if`, loops kabi mavzular chuqur emas. Rustga xos mavzular esa batafsilroq berilishi kutiladi. Shu sababli bu source wiki'dagi Rust Book materiallari bilan bir xil rol o'ynamaydi: u beginner foundationdan ko'ra back-end developer onboarding xaritasini beradi.

Kitob o'quvchidan oldindan quyidagilarni kutadi: arithmetic/logical operations, imperative constructs, data structures, basic algorithms, [[stack-and-heap|stack va heap]], console basics, [[threads|multithreading]], synchronization, [[http|HTTP]], relational databases, va JSON.
