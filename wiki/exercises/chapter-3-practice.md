# Chapter 3 Practice

## Goal

Chapter 3dagi variables, data types, functions, comments, va control flow conceptsni kichik programs orqali mustahkamlash.

## Exercises

1. Fahrenheit/Celsius converter yozish.
2. nth Fibonacci number generator yozish.
3. "The Twelve Days of Christmas" lyrics generator yozish, repetitiondan foydalanish.

## Hints

- Converter uchun function return value va numeric typesdan foydalaning.
- Fibonacci uchun `for` yoki `while` loop ishlating.
- Lyrics generator uchun arrays, `for` loop, va conditionalsdan foydalaning.

## Review Notes

- Har bir exercise uchun `cargo new` project ochish shart emas; bitta practice project ichida functions bilan ajratish mumkin.
- Intermediate values uchun default immutable variablesni afzal ko'ring, faqat counter/state kerak bo'lsa `mut` ishlating.
- Agar `if` expressionni `let`da ishlatsangiz, barcha arms bir xil type qaytarishi kerak.
