# 22.4. D - Useful Development Tools

## Learning Goal

Rust'ning rasmiy daily-development tooling qatlamini tushunish: formatting, automatic warning fixes, linting, va IDE integration.

## Main Ideas

- `rustfmt` community style'ni avtomatlashtiradi.
- `cargo fix` / `rustfix` compiler suggestion'larini qo'llab, warning fix'larni tezlashtiradi.
- [[clippy]] common mistake va quality lint'larini beradi.
- [[rust-analyzer]] IDE feedback loop'ni autocompletion, jump-to-definition, va inline errors bilan kuchaytiradi.

## Exercises

1. `cargo fmt`, `cargo fix`, va `cargo clippy`ni bitta project workflow'i ichida tartiblang.
2. `cargo fix` qaysi turdagi warninglarda yaxshi ishlashini yozing.
3. `rust-analyzer` compiler-centric tool deganda nima nazarda tutilishini tushuntiring.
