# Rust Programming Language LLM Wiki

This project is an Obsidian-friendly Rust learning wiki built for AI-assisted study. It was inspired by Andrey Karpathy's LLM Wiki idea and adapted for people learning Rust.

The goal is simple: turn Rust books, notes, examples, questions, and summaries into a durable knowledge base instead of scattered files. The wiki is still evolving.

## What This Project Is

- A personal Rust learning wiki
- A place to store source summaries, concept pages, examples, exercises, and questions
- A structure for learning Rust with AI while keeping the notes readable by humans

Most explanations and notes are in Uzbek. Standard Rust terms stay in English.

## Who It Is For

- People learning Rust
- People who want to study Rust with AI support
- People who prefer a markdown-based knowledge base over scattered notes

## How To Use

1. Open the project as an Obsidian vault.
2. Start with `wiki/index.md`.
3. Use `wiki/overview.md` for the current big picture.
4. Read source material in `raw/` and the processed learning pages in `wiki/`.

Opening the project in Obsidian is the most convenient way to use it.

## Project Structure

- `raw/` - original source material such as books, notes, articles, transcripts, and papers
- `wiki/` - the main knowledge base with summaries, concepts, chapters, examples, questions, and tools
- `output/` - generated reports and analysis files
- `infranodus/` - ontology and relation files
- `todos/` - learning and research workstreams

## Current Status

This project is a work in progress. It will continue to grow as more Rust sources are processed, linked, and refined.
