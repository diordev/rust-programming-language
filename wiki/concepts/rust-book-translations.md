# Rust Book Translations

## Short Definition

Rust Book'ning inglizchadan tashqari community translation ro'yxati uchun reference sahifa.

## Why It Matters

Language accessibility learning adoption'ni kuchaytiradi. Appendix F global translation ecosystem borligini ko'rsatadi.

## Mental Model

Bu live registry emas, source snapshot. Translation completeness va URL holati vaqt o'tishi bilan o'zgarishi mumkin.

## Syntax and Examples

Snapshotdagi alohida qayd:

- O'zbek: `rust-lang-uz/book`

## Common Mistakes

- Snapshot ro'yxatni current verified status deb o'ylash.
- Translation repository bor ekan, mazmun to'liq yoki up-to-date deb xulosa qilish.
