# 22.7. G - How Rust is Made and Nightly Rust

## Learning Goal

Rust release train modeli, stable/beta/nightly kanallari, unstable feature policy'si, rustup toolchain switching, va RFC governance jarayonini tushunish.

## Main Ideas

- Rust guiding principle: [[stability-without-stagnation]].
- Uchta release channel bor: [[release-channels]] orqali stable, beta, nightly.
- Unstable features faqat [[nightly-rust]] va [[feature-flags]] bilan ishlaydi.
- [[rustup]] project-level toolchain switchingni oson qiladi.
- Yangi language featurelar [[rust-rfc-process]] orqali muhokama, implementatsiya, va stabilizationdan o'tadi.
