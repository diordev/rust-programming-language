## Detailed Summary

Bu sahifa [[0-the-rust-programming-language|The Rust Programming Language]] kitobining title page qismi. U kitob qaysi Rust versiyasi va edition kontekstida o'qilishi kerakligini belgilaydi.

Manbaga ko'ra, bu matn Rust `1.90.0` yoki undan yangi versiyani asos qiladi. Rust `1.90.0` release sanasi `2025-09-18` deb berilgan. Kitobdagi projectlar `Cargo.toml` ichida `edition = "2024"` bilan Rust 2024 Edition idiomlariga mos yozilishi kutiladi.

Sahifa o'quvchini Rustni o'rnatish yoki yangilash uchun Chapter 1 dagi Installation bo'limiga, editionlar haqida tushuncha uchun Appendix E ga yo'naltiradi.

Kitob rasmiy HTML formatda `https://doc.rust-lang.org/stable/book/` manzilida mavjud. `rustup` orqali o'rnatilgan Rust bilan offline ochish ham mumkin: `rustup doc --book`.

Sahifa community translations borligini aytadi, shuningdek No Starch Press orqali paperback va ebook formatlarini eslatadi. Interaktivroq o'rganish uchun Brown versiyasi ham ko'rsatilgan: `https://rust-book.cs.brown.edu`.
