# RFC3339

## Short Definition

Internet date/time formatlaridan biri; chrono `DateTime` uchun `to_rfc3339()` beradi.

## Why It Matters

JSON APIs va logs uchun timezone offsetli timestampni aniq text formatda uzatish kerak.

## Mental Model

RFC3339 timestamp date, time, va offsetni bitta stringda saqlaydi, masalan `2025-12-15T13:51:10+00:00`.

## Syntax and Examples

```rust
let s = chrono::Utc::now().to_rfc3339();
```

## Common Mistakes

- Offsetni tashlab yuborib, timestampni ambiguous qilish.
- Custom format kerak bo'lmagan joyda qo'lda format string yozish.
