# Rust for Backend Developers: Setup Rust

## Learning Goal

Rust development environment faqat Rust compiler emas, balki linker va native toolchain bilan bog'liqligini tushunish.

## Main Ideas

- Rust source [[rustc]] orqali object codega compile bo'ladi.
- Final executable uchun [[linker]] object files, Rust artifacts, va C standard library qismlarini ulaydi.
- [[rustup]] Rust toolchainni boshqaradi; C/C++ toolchain esa OSga bog'liq alohida qatlam.
