# Edition

## Short Definition

`edition` Rust projectining language compatibility settingi.

## Why It Matters

Edition yangi language changesni projectlarga boshqarilgan tarzda kiritadi.

## Mental Model

Compiler version toolchainni bildiradi; edition esa source code qaysi Rust language edition qoidalari bilan o'qilishini bildiradi.

## Syntax and Examples

```toml
edition = "2024"
```

## Common Mistakes

- Editionni dependency version yoki compiler version bilan aralashtirish.
