# RFC2822

## Short Definition

Email/header-oriented date/time text format; chrono `DateTime` uchun `to_rfc2822()` beradi.

## Why It Matters

Ba'zi legacy protocols yoki headers RFC2822 style timestamp kutishi mumkin.

## Mental Model

RFC2822 human-readable weekday/month names va numeric offset bilan ishlaydi.

## Syntax and Examples

```rust
let s = chrono::Utc::now().to_rfc2822();
```

## Common Mistakes

- JSON API timestamp uchun RFC2822ni default tanlash.
- RFC3339 va RFC2822 outputlarini bir xil contract deb qabul qilish.
