# Raw Sources

This folder contains immutable source material.

Organize by source type:

- `books/` for book chapters and chapter notes.
- `notes/` for personal notes.
- `articles/` for web articles.
- `youtube/` for transcripts.
- `papers/` for papers converted to markdown.
- `assets/` for original PDFs, images, and binaries.

The LLM reads files here but does not rewrite them during processing.
