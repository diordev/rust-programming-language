# VSCode

## Short Definition

VSCode - extensionlar orqali Rust IDE workflowini beradigan code editor.

## Common Commands

```shell
cargo new test_rust_project
```

## Notes

- Rust support uchun `rust-analyzer` extension asosiy requirement.
- TOML files uchun Even Better TOML kabi extensionlar foydali.
- Smoke test: Cargo project ochiladi, `src/main.rs`dagi Run action Hello Worldni ishga tushiradi.
