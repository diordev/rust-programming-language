# 0.2. Introduction

## Learning Goal

Rust Bookni qanday o'qish kerakligini, Rust qaysi muammolarni hal qilishga urinayotganini, va keyingi chapterlar qanday learning path berishini tushunish.

## Main Ideas

- Rust faster va more reliable software yozishga qaratilgan.
- Rust high-level ergonomics va low-level control o'rtasidagi trade-offni kamaytirishga urinadi.
- Compiler Rustda gatekeeper: subtle bugs, ayniqsa concurrency bugs, compile bosqichida ushlanishi mumkin.
- Rust ecosystem productivity uchun [[cargo|Cargo]], [[rustfmt]] va IDE integration kabi toolarga tayanadi.
- Kitob sequential o'qishga mo'ljallangan, lekin learner ehtiyojiga qarab chapterlarni sakrab o'qish mumkin.
- Compiler error messagesni o'qish Rustni o'rganishning alohida ko'nikmasi.
